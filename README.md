# 🎓 School Management Dashboard (Excel)

## 📌 Project Overview

This project is an interactive **School Management Dashboard** created using Microsoft Excel.
It helps in managing and visualizing school data such as students, teachers, and academic activities in a structured and user-friendly way.

## 🚀 Key Features

* 📊 Interactive Dashboard Design
* 🎓 Class-wise Student Data (Class 6–10)
* 👩‍🏫 Faculty Section with Subject-wise Teachers
* 📅 Event & Activity Tracker (Completed / Pending)
* 📈 Monthly Activity Analysis with Charts
* 🎯 Class-wise Toppers & Overall Top 3 Students
* 🔍 Pivot Tables & Slicers for Dynamic Filtering

## 🛠 Tools & Techniques Used

* Microsoft Excel
* Pivot Tables
* Charts & Graphs
* Slicers
* Data Formatting
* Conditional Formatting

## 📊 Dashboard Sections

* 🏫 School Overview (Total Students, Teachers, Classes)
* 👩‍🏫 Our Faculty (Teacher Profiles with Subjects)
* 🎯 Class-wise Toppers
* 🏆 Top 3 Students of the School
* 📅 Event Status Tracker
* 📊 Completed vs Pending Activities Chart

## 📁 Dataset Details

* Student data for multiple classes (6 to 10)
* Subject-wise marks and percentage calculation
* Teacher and subject mapping
* Event records with due dates and status

## 💡 Learning Outcomes

Through this project, I learned:

* How to design interactive dashboards in Excel
* Data visualization techniques
* Using Pivot Tables and Slicers effectively
* Structuring real-world datasets
